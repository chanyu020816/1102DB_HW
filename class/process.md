*Install express generator*
```
$npm install express-generator -g 
```
**Generator app**
```
$ mkdir (folder name)
$ cd (folder name) 
$ express --view=ejs crown2_12(app name)
$ code crown2_12 (app name)
```
**Install package**
```
$ npm i
```
**Start app**
```
npm start
```
**Modify to nodemon**
```
$ npm install nodemon
```
